How to set up Cloudinary

1

Sign up free at cloudinary.com

2

Your Cloud Name is on the dashboard homepage

3

Settings → Upload → Upload Presets → Add preset → Signing Mode: Unsigned → Allowed formats: leave BLANK → Save

4

Click Test Connection above to verify before saving

5

For cloud deletion: go to Cloudinary → Settings → API Keys → copy your API Key and Secret into the fields above

Changing credentials will automatically migrate all your encrypted media to the new cloud. Keep this tab open during migration.
